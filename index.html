Skills

Jak vytvořit kvalitní skill

Soubor .claude/commands/nazev.md s jasnou strukturou:

# /review-copy

## Cíl
Zkontroluj texty v projektu.

## Proces
1. Načti všechny textové soubory
2. Zkontroluj překlepy a gramatiku
3. ⏸ Ukaž nálezy uživateli
4. Po schválení oprav

## Výstupní formát
- Soubor → řádek → problém → návrh

## Rules
- České uvozovky „takto"
- Max 2 věty na odstavec

## Acceptance test
- Žádný překlep neprojde

Agenti

Jak aktivovat agenty

Claude spustí agenty automaticky u složitějších úkolů. Pro vynucení použijte explicitní instrukce:

# Explicitní aktivace agentů:

"Spusť paralelně tyto úkoly:
 1. Agent: Vezmi design z Figma a vytvoř React komponentu
 2. Agent: Projdi existující kód a najdi reusable komponenty
 3. Agent: Zkontroluj přístupnost celé stránky
 Počkej na všechny výsledky a slož je dohromady."

# Klíčové slovo: "paralelně" / "spusť agenty"
# Claude rozpozná a rozdělí práci automaticky

Čím přesněji popíšete jednotlivé úkoly, tím lépe agenti pracují.
