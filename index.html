Začínáme

Instalace Claude Code

Vše probíhá v Terminálu na Macu. Návod: docs.anthropic.com/claude-code

# 1. Nainstalujte Claude Code
npm install -g @anthropic-ai/claude-code

# 2. Přejděte do projektu
cd ~/muj-projekt

# 3. Spusťte
claude

Potřebujete Node.js (v18+) a Anthropic účet s API klíčem.

Skills

Uložené příkazy pro opakované úkoly

Skills jsou šablony instrukcí, které Claude spustí jedním příkazem. Jako zkratky pro složitější workflows.

Agenti

Jak Claude zpracovává úkoly

Bez agentů

Claude zpracovává vše sekvenčně, krok za krokem. Jeden úkol najednou.

1. Přečti Figma
2. Najdi komponenty
3. Generuj kód
4. Zkontroluj A11Y
= jeden po druhém
vs

S agenty

Claude spustí více agentů paralelně. Každý řeší jiný úkol současně.

Agent 1 → Figma + kód
Agent 2 → Hledá komponenty
Agent 3 → Kontroluje A11Y
= vše najednou

Kontext

Claude + Notion

Notion slouží jako zdroj kontextu a paměť mezi sessions. Claude si před prací ověří kontext a na konci uloží závěry.

Notion
  • Před každým dotazem ověří, že má aktuální kontext
  • Stáhne si relevantní stránky z Notion
  • Na konci session uloží závěry a rozhodnutí zpět

Propojení přes Notion MCP server. Claude pak pracuje s vaší dokumentací jako s vlastní pamětí.
