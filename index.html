Instalace Claude Code

Vše probíhá v Terminálu na Macu. Návod: docs.anthropic.com/claude-code

# 1. Nainstalujte Claude Code
npm install -g @anthropic-ai/claude-code

# 2. Přejděte do projektu
cd ~/muj-projekt

# 3. Spusťte
claude

Potřebujete Node.js (v18+) a Anthropic účet s API klíčem.

📁 .claude/
  📁 commands/
    📄 review-copy.md
    📄 check-a11y.md
    📄 generate-og-image.md

Příklad: review-copy.md

# Kontrola copy

Projdi všechny texty v tomto projektu a zkontroluj:
- Překlepy a gramatiku
- Konzistenci terminologie
- Správné uvozovky „české"
- Delší texty rozděl na kratší odstavce

Výstup: seznam nalezených problémů s návrhem oprav.

Pak stačí napsat /review-copy v Claude Code a skill se spustí.

Agenti

Jak agenty v Claude Code využít

Claude sám rozhodne kdy použít agenty. Stačí popsat komplexní úkol a Claude rozdělí práci.

# Příklad promptu který spustí agenty:

"Vezmi design z Figma, vytvoř z něj React komponentu,
 přidej dark mode variantu a zkontroluj přístupnost.
 Výsledek commitni do nové větve."

# Claude automaticky:
# → Agent 1: Čte Figma design
# → Agent 2: Hledá existující komponenty v kódu
# → Agent 3: Kontroluje přístupnost
# → Hlavní Claude: Koordinuje a skládá výsledek

Agentic model se aktivuje automaticky při složitějších úkolech. Čím přesněji popíšete co chcete, tím lépe agenti pracují.
